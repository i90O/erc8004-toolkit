# Contributing to ERC-8004 Toolkit

Thanks for your interest! Here's how to contribute.

## Getting Started

```bash
git clone https://github.com/i90O/erc8004-toolkit.git
cd erc8004-toolkit
npm install
node src/cli.js stats  # Verify it works
```

## Pull Requests

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Test manually: `node src/cli.js <command>`
5. Commit with a clear message
6. Push and open a PR

## Code Style

- ESM only (`import`/`export`)
- Descriptive variable names
- Error handling on all chain calls
- Comments for non-obvious logic

## Ideas

Check the README for the contribution ideas list, or open an issue to discuss new features.
