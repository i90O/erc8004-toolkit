# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this toolkit, please report it responsibly:

1. **DO NOT** open a public issue
2. Email: kk@i90o.dev
3. Include: description, steps to reproduce, potential impact
4. We will respond within 48 hours

## Scope

- Vulnerabilities in the toolkit code
- Issues with the audit/scoring logic that could produce false negatives
- Data exposure risks

## Out of Scope

- Vulnerabilities in the ERC-8004 smart contract itself (report to the contract deployer)
- Issues with third-party dependencies (report upstream)
- Social engineering

## Disclosure

We follow responsible disclosure. We will:
1. Acknowledge receipt within 48h
2. Provide a fix timeline within 7 days
3. Credit you in the changelog (unless you prefer anonymity)
